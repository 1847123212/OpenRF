U8 InitializeRadioAPI()

Initialize the RadioAPI. On return, the radio hardware will be configured and will be asleep.

Parameters: none

Description: Resets the RadioAPI and any radio related microcontroller resources to a known state. Interrupts are disabled if the function returns kSuccess.

Result: kSuccess - Radio is in IDLE READY Mode kXTALError - Cannot start the radio's crystal oscillator kSPIError - SPI not working kUnsupportedRadioError - Wrong chip version... Doesn't work with our code

Parameters
dataRateChannel spacing between FHSS channels Data rate for TX and RX

tPacketTypes  packetType,

U8 xdata *  txBuffer

MethodResult SendPacket(U8 length, U8 *SDU)

Send a packet using the radio's built in packet engine.

Parameters: destAddress, unsigned long, destination for this message packetType, PacketTypes, type of packet (uniack, uninoack, multi) length, unsigned char, length of SDU SDU, unsigned char *, service data unit to send

Description: Calling this function will configure the radio to transmit a packet, load the FIFO and packet header information, and start the packet transmission. Once the process is started, this function will return. When the packet has been successfully sent, RadioNotifyTxPacketComplete() will be called. If an error occurs sending te packet, RadioNotifyTxPacketError() will be called. If the packet type is uniack, the transmit complete notification happens after an acknoweldgement is received. If the packet type is not uniack, the transmit complete notification will happen once the transmission finishes.

Returns: kSuccess

Parameters
packetTypeDestination MAC address
lengthPacket type
txBufferLength of packet SDU (service data unit or payload) Pointer to buffer containing packet SDU (service data unit or payload)

void SetTxPower ( U8  power)

U8 SetTxPower(U8 power)

Sets the radio transmit power. If the RFIC is a 1231H, the PA Boost will automatically be used for the high power setting.

Parameters: power,U8,Power setting. Varies based on chip. Masked with 0x07 prior to writing.

Description: Set the transmitter output power. Does not put the chip in TX mode, it just sets the TX power.

power Si1000/4432 Si4431 ===== ============== ================ 0 1 -8 1 2 -5 2 5 -2 3 8 1 4 11 4 5 14 7 6 17 10 7 20 13

Returns: Always returns kSuccess

Parameters
powerDesired power level. Bits 6,7,8 turn on PA0,PA1,PA2 respectively. Bits 0-4 set power level in 1dB increments. See 3.4.6 in SX1231 datasheet
